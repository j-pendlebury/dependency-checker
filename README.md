# Dependency Checker - WIP

Looks for pinned dependencies in package.json of all morph_modules.

Use Node6 or above.

# Usage
```
> git clone git@github.com:jblaut/dependency-checker.git
> cd dependency-checker
> npm install
> npm link
> cd path/to/morph-modules
> depchecker

```
